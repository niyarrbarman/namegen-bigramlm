# NameGen using Bigram Language Model

This is inspired by [Karpathy's makemore](https://github.com/karpathy/makemore).
The outputs are not of very high quality. This is just an experiment.

### Sample outputs
````
Input char is :  z
Outputs:
zaviah

Input char is :  d
Outputs:
demabe

Input char is :  m
Outputs:
manana
````
